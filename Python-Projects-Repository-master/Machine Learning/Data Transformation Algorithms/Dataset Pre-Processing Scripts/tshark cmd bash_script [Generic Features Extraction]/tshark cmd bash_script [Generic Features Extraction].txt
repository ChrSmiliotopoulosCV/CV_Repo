###The command that follows reads the pcap file, filters the pcap file based on the user's selection of generic features, separates the selection with comma and extract the selection to a .csv file with the headers included.###

tshark -r C:\Users\chrsm\Desktop\DeauthSelection.pcap -T fields -E separator=, -E header=y -e wlan.fc.type -e wlan.fc.subtype -e frame.time_delta -e frame.time_relative -e ip.src -e ip.dst -e ip.proto > C:\Users\chrsm\Desktop\DeauthSelection.csv

### tshark command to decode the encrypted wpa2 pcap file and extracted to a new decrypted one.###
